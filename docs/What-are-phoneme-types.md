# 音素的类型

- AFF  
  affricate 塞擦音  
  **c**hina **j**ust つ**ts**u 只**zh**i
  
- ASP  
  aspirate 送气音  
  **h**igh ひゃ**hy**a 和**h**e

- BRE  
  breath 呼吸音

- CDA  
  coda 尾音  
  还ha**i** 而e**r** 安a**n**

- DPT  
  diphthong 双元音  
  ab**ou**t b**i**te 小x**iao**
  
- FRI  
  fricative 摩擦音  
  **th**e ふぁ**f**a 西**x**i

- LIQ  
  liquid 流音  
  **l**ie ら**r**a 了**l**e

- LBR  
  long breath 长呼吸音

- NSL  
  nasal 鼻音  
  si**ng** にゃ**ny**a 末**m**o

- SVW  
  semivowel 半元音  
  **r**isk わ**w**a 亚**y**a

- SLI  
  slience 静音

- STP  
  stop 爆破音  
  bu**tt**er か**k**a 波**b**o

- VWL  
  vowel 元音  
  b**u**t い**i** 与y**v**
